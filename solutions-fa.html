RTMS Realtime Monitoring System

RTMS는 설비 및 공정 상태를 실시간으로 모니터링 하면서 제로 또는 낮은 대기 시간으로 지속적으로 업데이트되는 정보 스트리밍을 제공합니다. 관리자가 심각한 문제를 즉시 식별 할 수있는 관련 및 현재 데이터의 지연 시간이 짧은 연속 스트림을 제공하여 사전에 대비하며, 데이터를 추적함으로써 조직은 추세와 성과를 공개하고 예측할 수 있습니다.

  • 역할

  • 설비 Status & Alarm 정보
  • 설비/작업자/공정 별 불량 현황
  • 공정간 데이터 연계성 분석
  • 실시간 분석 데이터 제공
  • Bottleneck 분석 자료 제공
  • 설비 가동률 제공

  • 기대효과

  • 지속적인 설비 관리로 설비 신뢰성 확보
  • 최저 불량률 실현
  • 데이터 수집으로 인한 불필요한 인력 양산 방지
  • 데이터 수집 시간 최소화
  • 장애 및 불량에 대한 신속한 분석
  • 데이터 처리 간소화

DMS Defect Management System

비전 검사 설비에서 검출한 불량에 대해서 별도의 설비 없이 가상으로 설비의 데이터와 이미지를 사용하여 불량을 선 판정하는 시스템으로 과검출 된 Defect을 DRS에서 작업자가 직접 판정하고, 검사 대상 Defect을 선별하여 비전 검사 설비의 Load를 줄여주고 물류 성능을 향상합니다. 별도의 설비가 없으므로, 실물이 움직이지 않고 판정하므로 빠른 판정이 가능합니다.

  • 빠른 판정으로 물류 Tact time 최소화
  • 다양한 불량 정보 수집, 가공 하여 분석 및 통계데이터 제공
  • 불량 검출력 향상 조치 예방에 도움
  • 직관적 UI 구성으로 사용자 편의성 증대
  • 상세 불량정보 이미지로 조회 가능
  • 통계 데이터 Excel 파일 저장 다양한 분석 활용 용의
  • Smart-IT Server의 학습Data를 활용하여 기존 DRS시스템 을 고도화 하여 Repair설비의 Load를 줄여주고 물류 성능을 향상시킴
  • 안정되고 저비용 확장 성이 보장된 무정지 “DMS” 시스템 구축
  • Smart-IT Server와 연동할 수 있는 유연한 인터페이스제공
  • 검증된 DRS시스템 제공
  • 고객 작업환경에 맞는 UI제공

DMS와 Runner 서버 간의 Data Interface 구성

  • 자동 판정 Process Alive/Dead Check (TCP/IP 통신 연결 상태 유무로 판단)
  • 자동 판정 Process 및 관리 Process 상태를 DMS Server에 보고 (해당 Process 상태를 모니터링 하기 위한 용도)
  • MES 에서 Image Download 및 저장
  • Download한 Image 경로를 자동판정 Process에 전달
  • 자동 판정 완료 결과를 DRS Server에 보고
  • Download한 Image를 Data Server에 Upload
  • SAN Switch 사용으로 인한 Data 전송 안정성 확보

CIM & GEMComputer Integrated Manufacturing

장비(EQ)와 상위 HOST간의 매개체 역할을 하며, EQP 내에서 여러 UNIT을 통합 운영 하는 시스템으로서 공정 정보 및 물류 정보를 상위로 보고 하고 HOST에서 공정 데이터를 전송 받아 장비 운영에 사용합니다. Recipe에 따른 다양한 검사 방법을 제공하고 검사에 필요한 다양한 요소와의 인터페이스를 제공 함으로써 검사 진행의 Main Control을 담당합니다.

  • LCD / OLED / 반도체 생산 공정에서 데이터 또는 이벤트의 상하교환을 위해 HSMS에 따른 통신을 준수토록 지정
  • 물류 장비 및 데이터를 처리하는 객체간 통신 프로토콜 준수로 인한 복잡함을 대체하기 위해 SCP CIM 도입
  • HSMS외 효율적인 검사 모듈 및 데이터 처리를 포함하고 상황에 맞는 이벤트 처리와 Status 보고 등의 복잡한 기능들을 구현
  • 통신을 포함 더욱 다양하고 효율적인 검사 및 장비 운용을 위해서 SCP CIM 의 역할이 지속적으로 중요시
  • MES와 Online 유지 및 Spec에 따른 자동화 보고
  • Equipment Data 관리
  • Recipe에 따른 다양하고 능동적 검사 방식 제공
  • 판정제어 기능에 따른 작업자의 판정 실수 예방
  • Flexible Port 기능으로 효율적이고 편리한 포트 관리 방법 제공
  • Flexible Pattern기능으로 패턴 검사의 Tact을 줄일수 있는 방법 제공
  • Blind Test와 연동해 작업자의 검사 역량 강화
  • 안정적인 실시간 데이터 백업 및 Restore기능 제공
  • 검사를 하기 위한 사전 데이터 수집
  • 검사 결과를 다양한 수집 Object로 전송해 정확한 검사와 다양한 모니터링을 위한 리소스 제공
